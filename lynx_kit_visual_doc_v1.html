九层 Agent 详解

① 意图识别
Claude Haiku 4.5 快速语义理解
输入：自然语言描述 → 输出：产品类型 + 置信度 + 核心功能
② 架构师
选择最优技术架构
输入：产品类型 + 复杂度 → 输出：技术栈 + 依赖 + 成本预估
③ 产品经理
拆解功能模块
输入：需求 + 架构 → 输出：模块列表 + 用户流程 + 数据模型
④ 设计师
生成 UI 方案
输入：产品需求 + 品牌偏好 → 输出：设计系统 + 布局方案
⑤ 前端开发
生成界面代码
输入：设计稿 + API 规范 → 输出：React/Next.js 代码
⑥ 后端开发
生成 API/数据库
输入：数据模型 + 业务逻辑 → 输出：tRPC API + Prisma Schema
⑦ AI 集成
集成 LLM/RAG/工具
输入：AI 功能需求 → 输出：模型配置 + Prompt + 工具调用
⑧ 测试修复
验证/修复 Bug
输入：代码包 → 输出：测试结果 + 修复补丁（L1/L2/L3）
⑨ 部署发布
上线/上架商店
输入：代码包 + 部署目标 → 输出：URL + 商店上架
修复策略（三层）
L1 静默修复
编译错误、类型错误、导入错误 → AI 自动修复，不打扰用户
L2 引导修复
逻辑错误、功能不符 → 向用户展示 A/B/C 选择题
L3 安全回滚
致命错误、多次失败 → 自动回滚到上一可用版本

AI 功能说明

AI 模型选型

Claude Sonnet 4.6（主力）
代码生成最强，上下文 200K，适合复杂产品开发
$3/1M input, $15/1M output
Claude Haiku 4.5（快速）
速度快成本低，适合意图识别、分类、简单任务
$0.8/1M input, $4/1M output
DeepSeek-V3（降级）
中文场景优秀，成本极低，高并发时降级使用
~$0.3/1M input
Embedding（RAG）
text-embedding-3-large，向量检索，知识库支持
$0.13/1M tokens
